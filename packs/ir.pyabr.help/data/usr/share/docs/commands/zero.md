# zero

### Usage
Make files empty in local drive an upload original file to cloud drive.

### Options
```
zero [filename] ...
```

### Example

- **Example 1**: 
- `foo.bar` file:
```
Welcome to foobar
```
- Upload `foo.bar` to cloud drive using `ziro`
```
zero foo.bar
```
- `foo.bar` file:
```

```
- Download `foo.bar` from cloud drive using `down`
```
down foo.bar
```
- `foo.bar` after `down`:
```
Welcome to foobar
```
